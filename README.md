# CatchCatchMind
>출제자의 그림을 보고 제시어를 맞추는 게임이고 방을 만들어 여러 사용자들과 플레이 할 수 있다.

## 기능
* 사용자 리스트 출력
* 로비, 게임 채팅 
* 방 관리 (방 리스트/방 생성/입장, 퇴장)
* 그림판
* 출제자 턴 이동
* 서버에서 단어 제공
* 밑그림 - 출제자
* 힌트 - 출제자 외 사용자 (첫 글자/초성)
* 점수 관리 
* 랭킹 
* 타이머
